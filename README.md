Welcome to the **Rock Paper Scissors** game! This is a fun, interactive web-based implementation of the classic hand game. I built this project to showcase my front-end web development skills using HTML, CSS, and JavaScript, making it a great addition to my internship portfolio.

## đźŚź Features

*   **Interactive Gameplay:** Play the classic Rock-Paper-Scissors game against a computer opponent.
*   **Dynamic Score Tracking:** The game keeps a continuous, live tally of both your score and the computer's score.
*   **Instant Visual Feedback:** 
    *   Winning moves display a vibrant green background message.
    *   Losing moves display a red background message.
    *   Draws return a neutral blue background message.
*   **Clean & Responsive UI:** A simple, intuitive, and centered layout that is aesthetically pleasing and focuses on the core gameplay experience.
